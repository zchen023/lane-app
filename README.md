# Lane UI Kit v0

Reusable React + TypeScript + Tailwind UI foundation for Lane 1.0.

## What this includes

- AppShell
- Sidebar
- SidebarNavItem
- TopBar
- MainCanvas
- Button
- SearchInput
- PageHeader
- MetadataChip
- MetadataRow
- Panel
- PanelHeader
- ContentFrame
- InfoCard
- Icon
- SourceDetailPage rebuilt from the GS reference page direction
- UI rules doc

## Run locally

```bash
npm install
npm run dev
```

## Build check

```bash
npm run build
```
